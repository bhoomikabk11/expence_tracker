# Expense Tracker Web Application

A simple and user-friendly Expense Tracker web application designed to help users manage their daily finances efficiently. The application allows users to add, categorize, and track expenses with a clean and responsive interface.

## Features

* Add and manage expenses
* Categorize transactions
* Track spending history
* Responsive and clean UI
* Easy financial management

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase

## How to Run the Project

1. Download or clone the repository:

```bash
git clone <repository-link>
```

2. Open the project folder:

```bash
cd expense-tracker
```

3. Install dependencies (if applicable):

```bash
npm install
```

4. Configure Firebase:

* Create a Firebase project in Firebase
* Add your Firebase configuration in the project files

5. Run the project:

```bash
npm start
```

OR

* Open `index.html` directly in the browser
* Use VS Code Live Server extension for better performance

## Purpose

This project was developed to simplify personal expense tracking and provide users with a convenient digital finance management platform.
